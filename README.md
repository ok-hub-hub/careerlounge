# careerlounge

転職市場全般 UGC プラットフォーム。転職エージェント・転職体験談・年収口コミを、実体験ベースで投稿・比較・共有する UGC コミュニティ。

- **ドメイン**：[careerlounge.jp](https://careerlounge.jp)（2026-05-16 取得、有効期限 2027-05-31）
- **ターゲット**：20-40 代の転職検討層（IT エンジニア・営業職・コンサル・ハイクラス・第二新卒など幅広く）
- **運営**：AI Maker Lab（個人運営、ok-hub-hub）

## プロジェクトステータス

**Phase 1（基盤構築、2026-05-16〜）**：雛形構築・法務ページ整備・カテゴリ構造定義の段階。

詳細企画は AI 会社内部の以下ドキュメントを参照：
- `02_部門アウトプット/P-028_転職エージェント比較メディア_企画書v1.md` — 旧 IT 特化スコープでの企画書 v1（2026-05-15）
- `02_部門アウトプット/P-028_記事1_30代ITエンジニア_転職エージェント比較ピラー.md` — IT エンジニア 30 代向けサブピラー記事（約 6,800 字、調整事項は frontmatter `relegation_adjustments_required` を参照）
- `02_部門アウトプット/P-027_P-028_UGC共通基盤_設計案_Codex_v1.md` — Astro + Supabase の技術設計（Codex 一次案）

兄弟プロジェクト：[kokolabo](https://github.com/ok-hub-hub/kokolabo)（P-027、美容全般 UGC プラットフォーム）。UGC 投稿フォーム・モデレーション基盤を共通化する設計。

## 当初スコープとの変更点

当初は「IT エンジニア 30 代向け転職比較メディア」として企画されたが、2026-05-16 オーナー戦略変更で「転職市場全般＋エージェント比較＋投稿版（口コミ UGC）」プラットフォームに拡張。旧 IT 特化スコープの記事はサブピラーとして活用、トップピラー（転職市場全般）は別途新規執筆。

## 技術スタック（予定）

- **フレームワーク**：Astro v5 + TypeScript
- **スタイル**：Tailwind CSS
- **ホスティング**：Cloudflare Pages
- **DB / 認証**：Supabase（v1 以降、kokolabo と共通スキーマで `site_code = careerlounge`）
- **CMS / 記事**：Astro Content Collections + Markdown（編集記事）、Supabase（UGC）

## 法令対応の前提

- **ステマ規制**（2023-10 施行）：転職エージェント紹介リンク（アフィリエイト）の表記必須。
- **YMYL**（転職領域）：「必ず年収が上がる」「絶対に転職できる」のような断定表現を抑制。年収口コミは時期・業界・職種・地域をセットで表示。
- **個人情報**：転職体験談で会社名・部署名・個人情報が混入しやすいため、投稿前モデレーションを厳格化。

詳細は ARCHITECTURE.md と上記企画書を参照。
